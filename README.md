# 🧑‍💻 Online Examination System

## 📌 Description
Online Examination System is a Spring Boot based backend project where user authentication, exam management and submission system is implement


---

## ✨ Features

- 🔐 User Authentication (Login / Register)
- 📝 Create Exam (Teacher)
- ❓ Add Questions
- 📤 Submit Exam (Student)
- 📊 View Submissions
- 👥 Role-based system (Admin / Teacher / Student)

---

## 🛠️ Tech Stack

- Java
- Spring Boot
- Spring Security
- Hibernate, Spring Data JPA
- Gradle
- MySQL
- React JS
